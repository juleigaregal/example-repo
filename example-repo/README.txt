# Inventory Project

## Project Name
Inventory Project

## Description
This project manages and tracks inventory items. It allows users to add, update, and remove items from stock, as well as view current inventory levels. The code is designed to help businesses or individuals maintain accurate records of their products efficiently.

## Features
- Add new inventory items with relevant details (name, quantity, price, etc.)
- Update existing inventory records
- Remove items from inventory
- View current stock levels
- Optional: Generate simple reports
